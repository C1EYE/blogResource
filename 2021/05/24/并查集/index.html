并查集

  • 并查集也叫作不相交集合（Disjoint Set）

并查集有2个核心操作

  • 查找（Find）：查找元素所在的集合（这里的集合并不是特指Set这种数据结构，是指广义的数据集合）
  • 合并（Union）：将两个元素所在的集合合并为一个集合

有2种常见的实现思路

  • Quick Find 查找（Find）

    操作 查找（Find） 合并（Union）
    时间复杂度 O ( 1 ) O ( n )

  • Quick Union 查找（Find）的时间复杂度： O ( lo g n ) ，可以优化至 O 𝛼 𝑛

    合并（Union）的时间复杂度： O ( lo g n ) ，可以优化至 O 𝛼 𝑛， α ( 𝑛 ) < 5 ， α ( 𝑛 ) < 5
